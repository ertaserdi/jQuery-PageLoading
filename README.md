# jQuery-PageLoading
Sayfa açılışlarına process desteği olan bir loading bar.


### Example

```javascript
pageLoading({
    barColor:'#79c4df',
    barTop:'100px',
    textTop:'200px',
    backColor:'rgba(245, 245, 245, 0.90)',
    backBarColor:'#dfe8ea',
    text:'Lütfen bekleyin... <b>%{process}</b>',
    textVisible:true,
    loadOut:true
});
```
